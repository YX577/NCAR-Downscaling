This folder has a few Jupyter notebooks for plotting the data, etc.

Inside: 

  *animations.ipynb* - A notebook with some code to create and save animations of daily animations.
  
  *model_metrics.ipynb* - A notebook that contains the code to create distributions and to plot interannual variation in domain-wide precipitation averages.
  
  *Plot_data.ipynb* - Combines probability and intensity data using a static probability threshold. Plots and saves this data.
  
  *Scatterplots.ipynb* - Creates scatterplots comparing observed and RCM data. The notebook could be easily edited to plot any of the model info against RCM or obs data.
